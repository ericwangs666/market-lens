上线方案

建议先用 Cloudflare Pages 或 Vercel 部署静态前端，再用一个定时任务每天生成 JSON 数据。页面读取最新 JSON 后自动刷新。

1接入合法行情源：Tushare / 聚宽 / Alpha Vantage / Polygon / IEX Cloud / Nasdaq Data Link
2每天收盘后生成 `daily/YYYY-MM-DD.json`，包含指数、板块、龙头股和研报链接
3盘中定时拉取自选股行情，触发阈值后通过邮件、Telegram、企业微信或浏览器通知提醒
4加登录、云端备忘录、自选股、告警历史和回看

数据合规提醒

东方财富、同花顺的页面数据适合个人查看和研究，不建议直接抓取后商用或公开分发。正式上线建议使用授权 API，并在页面标注延迟行情和来源。
